Documentación estilos CSS

Por el Centro de Investigación en Tecnologías para la Sociedad - UDD

Esta documentación describe el propósito de las clases y estilos disponibles en el archivo CSS personalizado.

1. Introducción

Este archivo CSS proporciona estilos personalizados para la aplicación, incluyendo variables CSS, colores de fondo, estilos de texto, tarjetas (cards), sombras, bordes y más. La siguiente documentación describe cómo utilizar estos estilos para mejorar la apariencia de la interfaz de usuario.

2. Variables CSS (Custom Properties)

El archivo define varias variables CSS dentro del selector :root, lo que permite utilizar esquemas de colores personalizados a lo largo de la aplicación.

:root {
    --customprimary: #ffffff!important;
    --customsecondary: #F4F5F7!important;
    --customdark: #2C2C2C;
    --light: #d7d7d9!important;
    --success: #3EDC85!important;
    --dos: #261501;
    --warning: orange!important;
}

Ejemplos de uso:

  • --customprimary: Se utiliza para definir el color primario de fondo o texto.
  • --customsecondary: Color secundario, utilizado principalmente como fondo de la aplicación.
  • --customdark: Color oscuro utilizado para el texto o fondos oscuros.
  • --success: Color verde utilizado para mostrar estados de éxito.

3. Estilos Globales

Estos estilos afectan a todos los elementos del sitio, estableciendo márgenes y rellenos predeterminados y definiendo el color de fondo de la página.

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    background-color: var(--customsecondary)!important;
}

4. Componentes de Diseño de la Aplicación

4.1. Contenedores Principales

Las clases .app-container y .content permiten organizar los elementos de la aplicación y ajustar la altura mínima de la pantalla.

.app-container {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}

.content {
    flex: 1;
}

4.2. Footer

La clase .footer aplica un estilo coherente al pie de página, incluyendo color de fondo, padding y alineación de texto.

.footer {
    background-color: var(--customprimary)!important;
    padding: 20px;
    text-align: center;
    width: 100%;
}

5. Colores de Fondo y Texto Personalizados

Estas clases permiten personalizar el color de fondo y texto utilizando las variables definidas en :root.

.bg-customprimary { background-color: var(--customprimary)!important; }
.text-customdark { color: var(--customdark); }
.bg-success { background-color: var(--success)!important; }

6. Tarjetas (Cards)

Las tarjetas utilizan clases como .card y .card-title para definir un estilo consistente con sombra, bordes redondeados y efectos de hover.

.card {
    background-color: white;
    border-radius: 15px;
    padding: 20px;
    box-shadow: 0px 4px 8px rgba(0, 0, 0, 0.1);
}

.card:hover {
    transform: translateY(-5px);
    box-shadow: 0px 8px 16px rgba(0, 0, 0, 0.2);
}

7. Sombras y Bordes

El archivo define varias clases para aplicar sombras y bordes personalizadas a los elementos.

.box-shadow {
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2), 0 6px 20px rgba(0, 0, 0, 0.19);
}

8. Alturas y Anchos Responsivos

Las clases como .vh-* y .min-vh-* se utilizan para definir alturas y anchos relativos a la pantalla del dispositivo (viewport).

.vh-100 { height: 100vh; }
.min-vh-50 { min-height: 50vh; }

9. Clases de Texto y Formato

Las clases de texto permiten personalizar el estilo de los encabezados y el formato de los textos. Ejemplo de clases: .header-text, .handwriting, y más.

.header-text { font-weight: 600; }
.handwriting { font-family: 'Parisienne', cursive; font-size: 3.5rem; }

10. Media Queries (Estilos Responsivos)

Los estilos responsivos permiten adaptar el diseño a diferentes tamaños de pantalla. Ejemplo:

@media (max-width: 576px) {
    p, h5 { text-align: center; }
}

11. Clases de Hover y Opacidad

Las clases .hover-dark y .opacity-* permiten cambiar el comportamiento de los elementos en función de la interacción del usuario y la visibilidad.

.hover-dark:hover { background-color: var(--customdark)!important; }
.opacity-3 { opacity: 0.6!important; }
